# Vue Fundamentals

A small project for learning Vue.js.

## Topics

- Vue instance and lifecycle
- Components and props
- Directives (`v-if`, `v-for`, `v-bind`, `v-on`)
- Event handling
- Computed properties and watchers
- Vue Router basics
- Vuex / state management basics

## Notes

This repository is intended for practicing core Vue concepts and building simple examples.
